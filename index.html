O que significa CSS?


CSS significa "Cascading Style Sheets" (folhas de estilo em cascata, em português). É uma linguagem de estilo utilizada para definir a aparência visual e a formatação de documentos HTML e XML. Com o CSS, é possível definir cores, fontes, tamanhos, posicionamento de elementos, entre outros aspectos visuais de uma página web. O CSS permite separar o conteúdo de um documento HTML da sua apresentação visual, o que facilita a manutenção e a atualização do código fonte.

Como se define uma regra CSS? Quais são os três elementos que a compõe?


Uma regra CSS é definida por um seletor, um conjunto de declarações e um bloco de código delimitado por chaves. Os três elementos que compõem uma regra CSS são:


Seletor: o seletor é o elemento HTML ou grupo de elementos HTML que será estilizado pela regra CSS. Ele pode ser um elemento HTML específico, como "p", "h1", "div", ou pode ser um conjunto de elementos, como ".classe" ou "#id".


Declarações: as declarações são as propriedades CSS que serão aplicadas aos elementos selecionados. Cada declaração é composta por uma propriedade e um valor separados por dois pontos. Por exemplo, "color: red;" define a cor do texto dos elementos selecionados como vermelha.


Bloco de código: o bloco de código delimitado por chaves {} contém o seletor e as declarações da regra CSS.

Explique e dê exemplo de seletores (tag, identificadores, classes).


Os seletores CSS são usados para selecionar elementos HTML com base em sua tag, classe, id ou atributos específicos. Existem diversos tipos de seletores disponíveis no CSS. Aqui estão alguns exemplos:


Seletor de tag: O seletor de tag é usado para selecionar todos os elementos com uma tag específica. Por exemplo, para selecionar todos os parágrafos em um documento, use o seletor p. Exemplo: p {color: blue;}


Seletor de classe: O seletor de classe é usado para selecionar elementos que possuem uma classe específica. Uma classe é definida no HTML usando o atributo class. Por exemplo, para selecionar todos os elementos com a classe "destaque", utilize o seletor .destaque. Exemplo: .destaque { background-color: yellow;}


Seletor de ID: O seletor de ID é usado para selecionar elementos com um ID específico. O ID é definido no HTML usando o atributo id. Por exemplo, para selecionar o elemento com o ID "cabecalho", use o seletor #cabecalho. Exemplo: #cabecalho {font-size: 24px;}


Seletor de atributo: O seletor de atributo é usado para selecionar elementos que possuem um atributo específico. Por exemplo, para selecionar todos os elementos que possuem um atributo href, utilize o seletor [href]. Exemplo: [href] {color: green;}


Seletor de filho: O seletor de filho é usado para selecionar um elemento que é filho direto de outro elemento. Por exemplo, para selecionar todos os parágrafos que são filhos diretos de um elemento div, use o seletor div > p. Exemplo: div > p {font-weight: bold;}


Esses são apenas alguns exemplos de seletores CSS, há muitos outros disponíveis para selecionar elementos HTML de maneira precisa e eficiente.

Quais as três formas de incluir CSS em uma página?


Existem três formas de incluir CSS em uma página HTML:


Incluir o CSS em uma seção style dentro do próprio documento HTML. Nesse método, o CSS é inserido diretamente no arquivo HTML, entre as tags 'style' e '/style'. Esse método é mais adequado para pequenos projetos em que o CSS é aplicado em poucos elementos.
Exemplo: dentro de 'head'
< style >
p {color: red;}
< /style >


Incluir o CSS em um arquivo externo. Nesse método, o CSS é colocado em um arquivo separado com a extensão .css, em seguida, é vinculado à página HTML usando uma tag 'link' no cabeçalho do HTML. Esse método é mais adequado para projetos maiores em que o CSS é aplicado em várias páginas.
Exemplo: dentro do 'head'
< link rel="stylesheet" href="estilo.css" >


Incluir o CSS inline. Nesse método, o CSS é adicionado diretamente no elemento HTML usando o atributo style. Esse método é menos recomendado, pois torna o HTML menos legível e mais difícil de manter.
Exemplo: dentro da tag 'p'
< p > style="color: red;">Este é um exemplo de parágrafo.< /p >


Essas são as três formas de inclusão de CSS em uma página HTML. É importante escolher o método mais adequado para o seu projeto, levando em consideração o tamanho, a complexidade e a facilidade de manutenção do código.

Explique a propriedade CSS border.


A propriedade CSS border é usada para definir o estilo, a largura e a cor da borda de um elemento HTML. A sintaxe básica da propriedade border é a seguinte:


border: largura estilo cor;


Largura é a largura da borda, especificada em pixels, pontos, em ou outra unidade de medida;


Estilo é o estilo da borda, que pode ser solid (linha sólida), dotted (pontilhado), dashed (tracejado), double (linha dupla), groove (entrecortado), ridge (relevo), inset (entalhado) ou outset (destacado);


Cor é a cor da borda, especificada em um nome de cor, um valor hexadecimal ou uma função rgb() ou rgba().


Além disso, a propriedade border também permite a definição de cada aspecto da borda separadamente, usando as seguintes propriedades relacionadas:
border-width: define a largura da borda;
border-style: define o estilo da borda;
border-color: define a cor da borda.

Por exemplo, para definir uma borda de 1 pixel de largura, sólida e preta em um elemento HTML, podemos usar o seguinte código CSS:

border: 1px solid black;

Ou, para definir a largura, o estilo e a cor da borda separadamente, podemos usar o seguinte código:

border-width: 1px;
border-style: solid;
border-color: black;

Como mudar a fonte utilizada?


Para mudar a fonte utilizada em um elemento HTML por meio do CSS, você pode usar a propriedade font-family. A propriedade font-family define a fonte usada para o texto dentro de um elemento HTML.


Existem algumas maneiras diferentes de especificar a fonte que você deseja usar. Aqui estão três maneiras comuns:
1. Especificando uma fonte padrão
Você pode especificar uma fonte genérica como sans-serif, serif, monospace ou cursive que será usada se a fonte específica não estiver disponível. Aqui está um exemplo que define a fonte padrão do corpo do documento para "Arial" e a fonte padrão para títulos para "Helvetica Neue":

body { font-family: "Arial", sans-serif; }
h1 { font-family: "Helvetica Neue", sans-serif; }

2. Especificando uma fonte específica
Você também pode especificar uma fonte específica usando seu nome. Por exemplo:

h1 { font-family: "Open Sans", sans-serif; }

3. Especificando uma fonte da web
Você pode usar a propriedade @font-face para especificar uma fonte que será baixada da web. Aqui está um exemplo que define a fonte "Open Sans" para ser baixada da Google Fonts:

@font-face {
font-family: 'Open Sans';
font-style: normal;
font-weight: 400;
src: url(https://www...)
format('woff2');
};

h1 { font-family: "Open Sans", sans-serif; }

Observe que, ao especificar uma fonte da web, você precisa fornecer a URL da fonte, bem como o formato em que a fonte está disponível (geralmente woff2, woff ou ttf). Além disso, é importante fornecer uma fonte genérica como backup caso a fonte da web não possa ser carregada.

Explique a propriedade color e background-color.


'color': Essa propriedade define a cor do texto dentro de um elemento HTML. É possível utilizar diferentes formatos para especificar uma cor, como nomes de cores, códigos hexadecimais ou valores RGB.

Se usar
h1 {
color: red;
}

Nesse exemplo, a cor do texto do cabeçalho (h1) será vermelha.


'background-color': Essa propriedade define a cor de fundo de um elemento HTML. Também é possível utilizar diferentes formatos para especificar uma cor.

Se usar
body {
background-color: #F5F5F5;
}

Nesse exemplo, o fundo do corpo da página terá a cor cinza claro (#F5F5F5 em código hexadecimal).

Qual a vantagem de usar um arquivo externo para as regras do CSS?


A principal vantagem é que oferece uma melhor organização e manutenção do código, além de uma melhoria no desempenho da página. Com a separação de conteúdo e estilo, é mais fácil gerenciar e atualizar o design da página. Por exemplo, se um site tiver várias páginas, todas as páginas podem compartilhar o mesmo arquivo CSS externo, o que significa que qualquer alteração feita no arquivo CSS afetará todas as páginas que utilizam esse arquivo. Além disso, os navegadores podem armazenar em cache o arquivo CSS externo, o que pode reduzir o tempo de carregamento da página em visitas subsequentes.
